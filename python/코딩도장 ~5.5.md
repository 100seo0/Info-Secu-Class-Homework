파이썬에서는 복소수(complex number)도 사용할 수 있다.

허수부는 숫자 뒤에 j를 붙인다(공학에서는 허수를 j로 사용)

두 실수를 복소수로 만들 때는 complex사용

```python
>>> complex(1.2, 1,3)
(1.2+1.3j)
```



강제로 실수로 만들려면

- float(숫자)
- float(계산식)
- float('문자열')

실수는 float 자료형이며 type에 실수를 넣어보면

```python
>>> type(3.5)
<class 'float'>
```
